A chave Pix da Conexão Virtual (CNPJ 06.906.723/0001-30) é dado sensível.

**Enviar SOMENTE em pedidos explícitos:**
- "me passa o pix", "qual a chave pix?", "como faço pra pagar?", "me envia os dados pra transferir", "qual o CNPJ pra transferir?"

**NÃO enviar (mesmo com palavra "pix"):**
- Problemas técnicos: "o pix não caiu", "pix fora do ar", "recebi pix estranho"
- Perguntas sobre método: "vocês aceitam pix?" → responder só "sim, aceitamos" e perguntar se quer os dados
- Menções genéricas no meio de outra conversa

**Em dúvida:** confirmar antes — "Você quer fazer um pagamento pra gente? Posso te passar a chave."
